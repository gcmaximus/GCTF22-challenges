Fun with Flags!

I love flags!
There's a website that generates random flags around the world, but it doesn't have many options...

http://<ip-address>/index.html

--------------- Flag ---------------
GCTF{y0u_L13d_t0_m3}

--------------- Step-by-Step Solution ---------------
1. Open the Inspect Element page (Ctrl-Shift-I)

2. Expand the <body> tag. In the <img> tag, change the 'src' attribute to be './img/flag6.png'

3. Hidden flag appear, congrats!
